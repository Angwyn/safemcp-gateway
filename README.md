## Description
SafeMCP Gateway is a zero-dependency security proxy designed to protect local environments from malicious payloads generated by AI agents using the Model Context Protocol (MCP). As developers increasingly rely on autonomous agents for local execution, this gateway intercepts and sanitizes traffic, neutralizing prompt injections, path traversals, and Server-Side Request Forgery (SSRF) attacks before they reach your system shell.

## Key Features
* **Zero-Dependency Architecture:** Built entirely on native modules to eliminate supply-chain vulnerabilities.
* **Token-Bucket Rate Limiting:** Dynamically prevents denial-of-service and rapid-fire terminal payload spam.
* **Path Traversal Blocking:** Restricts AI file-system access strictly to authorized workspace directories.
* **SSRF Protection:** Sanitizes outbound network requests to prevent unauthorized internal network scanning.
* **Detailed Audit Logging:** Maintains an isolated `.jsonl` trail of all intercepted anomalous behaviors.
